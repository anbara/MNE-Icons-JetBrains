# MNE Icons JetBrains

<!-- Plugin description -->
This plugin adds the MNE Icons to the JetBrains IDEs.

Supported extensions:
- -raw.fif
- -epo.fif
- -ave.fif
- -tfr.h5

Note that these icons are illustrated by anbara and are not the official MNE Icons.
<!-- Plugin description end -->
